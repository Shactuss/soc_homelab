# SOC Homelab Project

## Overview
Building a home Security Operations Center lab to learn threat detection, log analysis, and incident response.

## Architecture
- Virtualization: VirtualBox
- Firewall: pfSense
- SIEM: Wazuh
- Attack Platform: Kali Linux

## Progress
- [x] Install VirtualBox
- [x] Set up pfSense firewall
- [x] Install Wazuh SIEM
- [x] Add vulnerable target machines
- [x] Configure log forwarding
- [x] Create detection rules
